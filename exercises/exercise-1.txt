Lab Exercise 1

Please copy and paste as much as you want from the demonstration

1. Create a new type called car type on the GraphQL server

2. Add the following fields

make
model
year
color
price

3. Create a connection type for the car type

4. Add the car type connection to the viewer type

5. Run the following query

query {
  viewer {
    cars {
      edges {
        node {
          id
          make
          model
          year
          color
          price
        }
      }
    }
  }
}
